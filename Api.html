Add Order

Parameters Description
key Your API key
action add
service Service ID
link Link to the post
quantity Quantity
custom Custom comments

Example

{
  "order": 1,
  "link": "https://www.instagram.com/p/123456",
  "quantity": 100,
  "custom": "Nice post!"
}
